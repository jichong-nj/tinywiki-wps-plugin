# 结构化文档编辑功能使用示例

## 功能概述

该功能包含两个主要部分：
1. **文档内容智能获取** - 自动获取文档内容发送给AI（选中文本优先，无选中则发送全文）
2. **结构化文档编辑** - AI在回复中包含结构化的文档操作指令，用户可以查看、确认并执行这些操作

## 文档内容智能获取功能

### 工作原理

当你在 WPS 文档中与 AI 对话时：
- **如果选中了一段文字**：系统会自动将选中的内容附加到你的问题中发送给 AI
- **如果没有选中任何文字**：系统会自动将整个文档的内容附加到你的问题中发送给 AI

### 用户界面

在聊天界面的输入区域上方，你会看到一个提示栏：
- 📝 使用选中内容 - 当有文本被选中时显示
- 📄 使用文档全文 - 当没有选中文本时显示

### 消息展示

当你发送消息后，你的消息气泡会显示：
- 你输入的原始问题
- 一个标签，指示是否包含了选中内容或文档全文

### 使用场景示例

#### 场景 1：修改选中的段落

1. 在文档中选中一个你想修改的段落
2. 在聊天框中输入："帮我润色这段文字，让它更正式"
3. 点击发送
4. 系统会自动将你选中的段落附加到问题中发送给 AI
5. AI 会基于选中的内容进行回复

#### 场景 2：分析整个文档

1. 不要选中文档中的任何内容
2. 在聊天框中输入："帮我总结一下这篇文档的主要内容"
3. 点击发送
4. 系统会自动将整个文档的内容附加到问题中发送给 AI
5. AI 会基于整篇文档进行回复

#### 场景 3：修正错别字

1. 选中包含错别字的句子
2. 输入："帮我检查并修正这段文字中的错别字"
3. 发送
4. AI 会分析你选中的内容并提供修改建议

## AI 回复格式

### 基本格式

AI 的回复应该是普通 Markdown 文本，其中包含一个用 ```document-operations 包裹的 JSON 数组。

```markdown
这是普通的回复文本，解释要做什么操作。

```document-operations
[
  {
    "type": "replace|insert|delete",
    "position": {
      "page": 1,
      "line": 10,
      "paragraph": 2,
      "context": "上下文文本",
      "contextStart": "起始文本",
      "contextEnd": "结束文本"
    },
    "oldContent": "原内容（delete 和 replace 需要）",
    "content": "新内容（insert 和 replace 需要）",
    "description": "操作描述"
  }
]
```

用户可以点击"执行"按钮来执行这些操作。
```

## 操作类型

### 1. 新增内容 (insert)

在指定位置插入新内容。

```json
{
  "type": "insert",
  "position": {
    "paragraph": 3,
    "contextStart": "前一段的结尾",
    "contextEnd": "后一段的开头"
  },
  "content": "这是要插入的新内容\n可以有多行",
  "description": "在第3段后插入新段落"
}
```

### 2. 删除内容 (delete)

删除指定的内容。

```json
{
  "type": "delete",
  "position": {
    "context": "包含要删除内容的段落"
  },
  "oldContent": "要删除的内容",
  "description": "删除冗余内容"
}
```

### 3. 替换内容 (replace)

用新内容替换旧内容。

```json
{
  "type": "replace",
  "position": {
    "page": 1,
    "paragraph": 1
  },
  "oldContent": "旧的表述方式",
  "content": "新的更好的表述方式",
  "description": "优化第一段的表述"
}
```

## 完整示例

### 示例 1：优化文档表述

```markdown
我已经分析了您的文档，建议进行以下优化：

1. 优化引言部分的表述
2. 补充实验数据
3. 调整结论

```document-operations
[
  {
    "type": "replace",
    "position": {
      "paragraph": 1,
      "context": "随着科技的发展"
    },
    "oldContent": "随着科技的发展，人工智能变得越来越重要",
    "content": "随着人工智能技术的快速发展，其在各个领域的应用变得越来越广泛和重要",
    "description": "优化引言第一段"
  },
  {
    "type": "insert",
    "position": {
      "paragraph": 5,
      "contextStart": "实验结果如下",
      "contextEnd": "综上所述"
    },
    "content": "\n根据最新的实验数据，我们的方法在准确率上提升了 15%，在召回率上提升了 12%。\n",
    "description": "补充实验数据"
  },
  {
    "type": "delete",
    "position": {
      "context": "这是一段冗余的内容"
    },
    "oldContent": "这是一段冗余的内容，需要删除掉",
    "description": "删除冗余段落"
  }
]
```

请查看以上操作，确认后点击执行按钮。
```

### 示例 2：修正错别字

## 用户界面功能

### 操作列表

- 显示所有 AI 建议的操作
- 每个操作显示：
  - 操作类型（新增/删除/修改，带颜色区分）
  - 操作描述
  - 位置信息（页码/行号/段落号）
  - 原内容和新内容的对比
  - 确认复选框

### 批量操作

- "全选"按钮：选中所有操作
- "全不选"按钮：取消所有选择
- "执行已确认的操作"按钮：执行所有选中的操作

### 定位功能

- 每个操作旁都有"📍 定位"按钮
- 点击后会自动跳转到文档中的对应位置

## 注意事项

1. **确保 WPS 文档已打开**：执行操作前需要先打开一个 WPS 文档
2. **确认操作**：建议先点击定位查看要修改的位置，确认无误后再执行
3. **操作顺序**：操作会按数组中的顺序依次执行
4. **文本匹配**：定位主要依赖文本内容匹配，请确保文档内容未被修改
